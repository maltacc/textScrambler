A little text scrambler. I was thinking of the order of letters in a word -- how we write a word from left to right (ex. w => o => r => d), but when we view those same characters on a keyboard from left to right, we end with a different word (w => d => r => o). This repo provides a small text editor to visualize words sorted in keyboard order.

To run: 
- `npm install`
- `npm run dev`
